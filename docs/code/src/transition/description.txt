A: Transition
  1. The transition property is an abbreviation for four properties: transition-property, transition-duration, transition-timing-function, and transition-delay.
  2. Supported properties and values: color, border, border-radius, transform, opacity, box-shadow, width, padding.
  3. Commonly used Bezier curve functions: ease, linear, ease-in, ease-out, ease-in-out.
